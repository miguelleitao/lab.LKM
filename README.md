# lab.LKM
Linux Kernel Modules lab handout 

# Howto
```
cp ok/mymodule0.c mymodule.c
make
modinfo mymodule.ko
insmod mymodule.ko
lsmod |head
dmesg
```

```
cp ok/mymodule1.c mymodule.c

dmesg
```
